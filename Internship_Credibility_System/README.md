# 🛡️ Internship Credibility Verification System

An advanced, AI-powered full-stack application designed to protect students and recent graduates from fraudulent internships and job scams. 

This system uses a **Hybrid Artificial Intelligence Pipeline**—combining a deep neural network (Transformer NLP) with a rigorous Rule-Based Engine—to analyze job postings in real time and deliver a **highly accurate Confidence Score** indicating whether the job is legitimate or a scam.

---

## 🌟 Key Features
- **Deep Learning NLP**: Uses `DistilBERT` (Fine-tuned Transformer) to understand the contextual language of job descriptions, easily outperforming standard keyword matchers.
- **Rule-Based Extortion Engine**: Instantly red-flags common scam tactics like "training module fees", "mandatory registration deposits", and generic free email accounts (`@gmail.com`). 
- **Explainable AI (XAI)**: Doesn't just give you a score. Automatically highlights the exact suspicious words inside the job description and lists logical observations explaining *why* the job was marked safe or dangerous.
- **Microsecond Inference**: Engineered strictly without heavy database lookups, relying on in-memory Tensor processing via `FastAPI` to deliver real-time sub-2-second evaluations.
- **Premium Glassmorphism UI**: Beautiful, interactive front-end designed entirely with Vanilla JavaScript, modern CSS animations, and gradient fluid designs.

## 🚀 Installation & Setup Guide

### 1. Clone the Repository 
*(Or extract the provided ZIP file onto your Desktop)*
```bash
cd "Internship_Credibility_System"
```

### 2. Install Python Dependencies
It is highly recommended to use a virtual environment (`venv` or `conda`).
```bash
pip install -r requirements.txt
```

### 3. Run the Backend AI Server
The FastAPI application handles the deep learning inference and rule processing. It must be running for the UI to predict accurately.

Navigate into the `backend` folder and start `uvicorn`:
```bash
cd backend
uvicorn main:app --reload
```
You should see output confirming the server has started on `http://0.0.0.0:8000`.

### 4. Run the Web Interface
Open a *new* terminal, navigate to the `frontend` folder, and start a simple HTTP server to avoid CORS/local file restrictions:
```bash
cd frontend
python -m http.server 3000
```
Navigate to **`http://localhost:3000`** in Google Chrome or Edge to view and use the live application!

## 🧠 Model Training (Optional)
If you wish to fully train the DistilBERT model weights yourself (requires a CUDA-compatible GPU for reasonable speed):
1. Navigate to `backend/model/`
2. Run `python train.py`
3. This will download the open-source `amarnanth/fake-job-posting-prediction` dataset from Hugging Face, tokenize it via `DistilBERT`, classify it into scam/legit labels, and save the resulting weights inside a new `saved_model` folder.
*Note: The system features an ultra-robust advanced NLP heuristic fallback allowing the UI to function flawlessly even if the heavy model weights haven't finished training.*
